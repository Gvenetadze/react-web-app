Project Description

SimpleShop is a simple and responsive web application built with React.
It demonstrates core React concepts such as routing, hooks, API integration, theming, multilingual support, and UI interactions like modals and animations.

The project was created as part of a React assignment.

Features

* 3 pages: Home, Products, About

* React Functional Components

* React Router for navigation

* React Hooks (useState, useEffect, useContext)

* API integration (Fake Store API using Axios)

* Modal window with animations

* LocalStorage usage

  Language preference

  Theme preference

* Responsive design (mobile, tablet, desktop)

* Light / Dark theme

* Multi-language support (English / Georgian)

* Clean project structure
